Selenium and RSS based Java job scraper. That's it ! What you do with that tool is up to you.

## Installing

### With Maven

You can add `io.makepad.jobfinder` in your `pom.xml` file as a dependency.

```xml
<dependency>
	<groupId>io.makepad</groupId>
	<artifactId>jobfinder</artifactId>
	<version>0.0.5</version>
</dependency>
```
